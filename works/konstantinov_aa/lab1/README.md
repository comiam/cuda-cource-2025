# Lab 1: CUDA Graphics - Вывод геометрических фигур

## Описание
Программа на CUDA, которая рисует круг в консоли используя символы ASCII.
Вычисления (определение принадлежности точки границе круга) выполняются параллельно на GPU.

## Использование
Программа запросит размеры поля (ширина и высота). Рекомендуемые значения: `60 30`.
Радиус круга рассчитывается автоматически как половина минимальной стороны.

## Лабораторная 1

### Описание задачи

Написать и скомпилировать программу на CUDA, которая выводит в терминал любую геометрическую фигуру (круг, квадрат, треугольник или комбинацию нескольких фигур).

Требования:
- Программа должна работать на GPU с использованием CUDA
- Вывод происходит в консоль (ASCII art)
- Фигура рисуется параллельно на GPU
- Код скомпилирован без ошибок и предупреждений

### Компиляция

```make```

### Запуск

```./main```

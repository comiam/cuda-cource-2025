# CUDA Graphics Lab

Программа для рисования геометрических фигур с использованием CUDA.

## Компиляция

```bash
make
```

## Запуск
```bash
./graphics [size] [shape]
```


## Примеры
```bash
./graphics 20 circle
./graphics 15 square
./graphics 15 triangle
```


## Поддерживаемые фигуры
- circle - круг
- square - квадрат
- triangle - треугольник

## Ограничения
- Максимальный размер: 200
- Минимальный размер: 5

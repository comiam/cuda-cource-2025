# Lab 1: CUDA Graphics

## Описание

Программа выводит в терминал геометрическую фигуру квадрат,
используя параллельные вычисления на CUDA.

## Требования

- CUDA Toolkit 11.0 или выше
- GPU с compute capability 7.0+
- Linux с GCC 9.0+
- Make

## Компиляция
```bash
nvcc -std=c++11 -o graphics.exe src/graphics.cu
```

## Запуск
```bash
./graphics
```

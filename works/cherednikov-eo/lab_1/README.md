# Lab 1: CUDA Graphics

## Описание

Программа выводит в терминал геометрическую фигуру (квадрат), используя параллельные вычисления на CUDA. Каждый поток GPU отвечает за один пиксель на канвасе, определяя, должен ли он быть частью границы квадрата.

## Требования

- CUDA Toolkit 11.0 или выше
- GPU с compute capability 7.0+
- Компилятор с поддержкой C++14
- Make (опционально, для компиляции)

## Компиляция

```bash
make
```

Или с указанием compute capability:
```bash
make COMPUTE_CAP=86
```

Альтернативно, прямая компиляция:
```bash
nvcc -O3 -arch=sm_75 -std=c++14 -o lab_1 main.cu
```

Очистка скомпилированных файлов:
```bash
make clean
```

## Запуск

```bash
./lab_1
```

Программа не принимает аргументы командной строки. Размеры квадрата и отступы заданы в коде:
- Ширина: 28
- Высота: 28
- Отступ от края: 2

- Размеры квадрата и отступы захардкожены в коде

- Для изменения размеров необходимо редактировать код и перекомпилировать

# Лабораторная N1

## Описание

Программа, которая рисует круг в терминале. 

## Компиляция

```bash
make
```

или:

```bash
nvcc -O2 -arch=sm_70 -o circle src/main.cu
```

## Запуск

```bash
./circle          # размер по умолчанию ~24
./circle 40       # больший круг
./circle 15       # маленький
```

## Особенности реализации

- 2D-сетка потоков (каждый поток = один символ)
- Коррекция пропорций символов терминала (dy × 2)
- Проверка ошибок CUDA через макрос CUDA_CHECK
- Поддержка аргумента размера из командной строки

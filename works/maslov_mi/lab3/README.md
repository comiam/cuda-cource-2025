# Lab 3: Sobel Operator - Оператор Собеля

## Описание задачи
Реализовать и оптимизировать оператор Собеля на CUDA для выделения границ на изображении.

---

## Требования
- CUDA Toolkit (`nvcc`) установлен
- `make` (для удобства сборки)

---

## Запуск и примеры

### Запуск

Сборка кода:

```bash
make
```
Запуск кода:

```bash
./bin/lab3 path/to/input_image.png
```

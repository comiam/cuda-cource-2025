## Lab 1: CUDA Graphics - Вывод геометрических фигур

### Описание задачи
Написать и скомпилировать программу на CUDA, которая выводит в терминал геометрическую фигуру (круг и квадрат).

### Требования
- ✓ Программа должна работать на GPU с использованием CUDA
- ✓ Вывод происходит в консоль (ASCII art)
- ✓ Фигура рисуется параллельно на GPU
- ✓ Код скомпилирован без ошибок и предупреждений

### Запуск

Сборка кода:

```bash
make
```
Запуск кода:

```bash
./lab1
```
